# 협력사 로고

로고 파일을 여기에 넣고 `src/data/partners.ts`의 해당 항목에 `logo` 경로를 채우면
워드마크(텍스트) 대신 이미지가 렌더된다.

```ts
{
  id: 'samsung-electronics',
  name: { ko: '삼성전자', ... },
  since: '1995',
  logo: '/images/partners/samsung-electronics.svg',  // 이 줄만 추가
}
```

## 권장 사양

- 형식: SVG 우선, 없으면 투명 배경 PNG (2x)
- 높이 기준 40px로 렌더되므로 세로 96px 이상 권장
- 단색(모노) 버전이 있으면 그쪽이 그리드에서 통일감이 좋다

## ⚠ 상표권

타사 로고는 각 사 상표다. 게시 전 해당 기업의 브랜드 가이드라인을 확인하고
필요하면 사용 허락을 받을 것. 납품 사실이 있어도 로고 사용권이 자동으로 생기지는 않는다.
현재 워드마크(회사명 텍스트) 방식은 이 리스크가 없다.
